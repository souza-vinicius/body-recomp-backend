# Frontend - Body Recomp App

Frontend application built with Next.js 14, React 18, TailwindCSS, and Capacitor.

## Setup

```bash
npm install
```

## Running Dev Server

```bash
npm run dev
```

## Testing

```bash
npm run test:unit
npm run test:integration
npm run test:e2e
```

## Capacitor Build

```bash
npm run build
npm run cap:sync
```
